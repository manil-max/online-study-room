# V8 Arşivi

release ve rollback belgelerini saklar.

yayın kararı veya açık iş kaynağı değildir.** Güncel kararlar için önce kökteki
`README.md`, `KALITE-PROGRAMI.md`, `PLAY-STORE-HAZIRLIK-TARAMASI.md` ve
repo kökündeki `progress.md` okunur.
