# Online Çalışma Sınıfı

YPT (Yeolpumta) benzeri, küçük gruplar için **ortak online çalışma uygulaması**. Kullanıcılar
aynı "sınıfa" katılır, birbirlerinin **canlı çalışma durumunu** görür, çalışma sürelerini
takip eder ve **detaylı istatistiklerle** kıyaslar.

## Özellikler (planlanan)

- 👥 **Sınıf/grup sistemi** — davet koduyla ortak çalışma odası
- 👤 **Profil** — e-posta ile hesap, profil fotoğrafı
- 🔴 **Canlı çalışma ekranı** — kim aktif çalışıyor, gerçek zamanlı (masa/lamba görünümü)
- ⏱️ **Süre takibi** — otomatik sayaç + manuel süre girişi
- 📊 **Detaylı istatistikler** — günlük ortalama, hafta içi/sonu, tarih aralıkları, kıyaslama
- 🔄 **Çoklu cihaz senkronizasyonu** — Android telefon/tablet + Windows
- 🧩 **Widget'lar** — Android ana ekran + Windows masaüstü

## Teknoloji

- **Flutter** (Android + Windows, tek kod tabanı)
- **Supabase** (Auth · Postgres · Realtime · Storage)
- **Riverpod** (durum yönetimi) · **fl_chart** (grafikler)
